# Agent Wiki

This wiki is compiled working context for agents editing this repo. Code, package config, workflows, and deck Markdown remain the source of truth.

Start here:

1. Read `AGENTS.md`.
2. Read `docs/agent/current-state.md`.
3. Read `docs/agent/repo-map.md`.
4. Check `docs/agent/open-questions.md`.
5. Use targeted `rg`, `find`, and file reads before editing.

After significant work, update:

- `docs/agent/current-state.md`
- `docs/agent/log.md`
- a new file under `docs/agent/session-handoffs/`
